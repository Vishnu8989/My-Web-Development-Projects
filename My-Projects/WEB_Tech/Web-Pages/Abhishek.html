Introduction

Lists are used to group together related pieces of information so they are clearly associated with each other and are easy to read. In modern web development, lists are workhorse elements, frequently used for navigation as well as general content.

Lists are good from a structural point of view as they help create a well-structured, more accessible, easy-to-maintain document. They are also useful because they provide specialized elements to which you can attach CSS styles.

Ordered List

An ordered list starts with the < ol> tag. Each list item starts with the li tag.The default value for the list items is marked as 1,2,3... and so on but we can modify them to some other type if we want.This leads us to the type attribute.The type attribute of the ol tag, defines the type of the list item marker

    Type                 Description

  < ol type="1">        The list items will be numbered with numbers (default)

  < ol type="A">        The list items will be numbered with uppercase letters

  < ol type="a">        The list items will be numbered with lowercase letters

  < ol type="I">        The list items will be numbered with uppercase roman numbers

  < ol type="i">        The list items will be numbered with lowercase roman numbers

  < ol type="1"   start=11>    The list items will be numbered with numbers starting from 11

  < ol type="A"   start=10>    The list items will be numbered with uppercase letters starting from J

     An example of ordered List:

< ol>
   < li>  OS < /li>
   < li>  COA < /li>
   < li>  WT < /li>
< /ol>

It will be displayed on the web page as given below

  1. OS
  2. COA
  3. WT

Unordered List

An unordered list starts with the < ul> tag. Each list item starts with the < li> tag. Here, no numbering is there.The list items will be marked with discs by default The type attribute of the ol tag, defines the type of the list item marker

    Value             Description

   < ul type="disc">      Sets the list item marker to a bullet (default)

  < ul type="circle">     Sets the list item marker to a circle

  < ul type="square">     Sets the list item marker to a square

   < ul type="none">     The list items will not be marked

     An example of unordered list :

< ul>
   < li>   pencil < /li>
   < li>   pen < /li>
   < li>   notes < /li>
< /ul>

It will be displayed on the web page as given below

  • pencil
  • pen
  • notes

Now suppose we write the html code as:
< ul type="square">
< li type="circle"> Tea < /li>
< li type="circle"> Coffee
< /ul>

Here, square will be overwridden by circle and the web page will look like:
  • Tea
  • Coffee

Description List

A description list is a list of items with a description or definition of each item. The description list is created using < dl> tag. The < dl> tag is used in conjunction with the < dt> tag which specify a term, and the < dd> tag which specify the term's definition.

The < dl> tag defines the description list, the < dt> tag defines the term (name), and the < dd> tag describes each term

Example:

< dl>
   < dt>Coffee < /dt>
     < dd>- black hot drink < /dd>
   < dt>Milk < /dt>
     < dd>- white cold drink < /dd>
< /dl>

It will be displayed on the web page as given below:   

Coffee
    
- black hot drink
  
Milk
    
- white cold drink
